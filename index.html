Acerca de mí

Profesional con 26 años de experiencia en proyectos principalmente del área TeleCom y Financiera. He participado en las distintas etapas del ciclo de vida del desarrollo y mantención de software desempeñando distintos roles: programadora, analista funcional, consultora de implantación, analista QA y líder de equipos.

Formación

Estudios de enseñanza básica y media en colegios municipales de Playa-Ancha, Valparaíso.

En el año 1992 ingreso a la carrera de Ingeniería Ejecución en Informática en la Universidad Católica de Valparaíso; titulándome el año 1996.

Actualmente me encuentro realizando el Bootcamp de desarrollo web en Desafío Latam.

Paquete Microsoft (word,excel,project,powerpoint,teams), Postman SoapUi,Visual Studio, Intellij Idea, Toad, Microsoft Sql Server Management, Bitbucket,Jira,IU Vision RPA,Jenkins,Cobol AS-400, Java 1.8 (spring Boot), .net Visual Basic, Visual Basic 6.0, C, ASP, PL/SQL Oracle (8,9i,10g),SqlServer, Git, Github, HTML, CSS, BootStrap.
